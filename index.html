A* Pathfinding Visualizer

Github link

Back to project list



  • Objective:

  • My pathfinding visualizer lets the user create barriers between the start and end point. Then my A* pathfinding algorithm will avoid the barriers and find the shortest path to the end point in real time.

  • Tech stack:

  • Python, pygame library

  • Classroom or self-taught:

  • I haven't taken a class that has covered pathfinding, so I had to teach myself.

  • Most interesting/challenging aspect:

  • Was learning about pathfinding algorithms. I chose A* over other popular pathfinding algorithms because it is flexible and efficient. Like Dijkstra's Algorithm, A star can find the shortest path between two points. However, A* is more efficient because, like a Greedy Best-First-Search, it uses a heuristic to guide itself.
